# 💰 FinançaLivre

> Aplicativo de controle financeiro pessoal — gratuito, instalável e seguro.

**Acesse agora:** https:///UmAprendizBrasileiro.github.io/financalivre

---

## ✨ Funcionalidades

| Módulo | O que faz |
|---|---|
| 🏠 **Início** | Saldo do mês, meta de economia, alertas automáticos |
| 💸 **Gastos** | Lançar entradas e saídas por categoria |
| 🏦 **Dívidas** | Controle de empréstimos com cálculo de parcelas e juros |
| 📊 **Relatórios** | Gráficos de gastos por categoria e tendência mensal |
| 📈 **Mercado** | Criptomoedas em tempo real + ações da B3 (via Brapi) |
| ⚙️ **Config** | Metas, alertas, cartão de crédito e token Brapi |

## 📲 Como instalar no celular

### Android
1. Abra o link do app no **Chrome**
2. Toque no banner verde **"Instalar FinançaLivre"** que aparece na tela
3. Confirme — o ícone aparece na tela inicial como app nativo

### iPhone / iOS
1. Abra o link no **Safari**
2. Toque no botão **Compartilhar** (□ com seta para cima)
3. Role e toque em **"Adicionar à Tela de Início"**
4. Confirme — pronto!

## 🚀 Como publicar no GitHub (passo a passo)

### 1. Criar conta no GitHub
- Acesse [github.com](https://github.com) e crie uma conta gratuita

### 2. Criar o repositório
- Clique em **"New"** (botão verde no canto superior esquerdo)
- Nome: `financalivre`
- Deixe **Public** marcado
- Clique em **"Create repository"**

### 3. Subir os arquivos
- Na página do repositório vazio, clique em **"uploading an existing file"**
- Arraste **todos os arquivos desta pasta** para a área de upload:
  - `index.html`
  - `manifest.json`
  - `sw.js`
  - `icon-192.png`
  - `icon-512.png`
  - `README.md`
- Clique em **"Commit changes"**

### 4. Ativar o GitHub Pages
- No repositório, clique em **Settings** (engrenagem)
- No menu lateral, clique em **Pages**
- Em **"Branch"**, selecione `main` e pasta `/ (root)`
- Clique em **Save**
- Aguarde 1–2 minutos

### 5. Acessar seu app
- O link será: `https://SEU-USUARIO.github.io/financalivre`
- Compartilhe esse link com qualquer pessoa — funciona em qualquer celular!

## 🔑 Ativar cotações de ações B3 (opcional)

1. Acesse [brapi.dev](https://brapi.dev) e crie conta gratuita
2. Copie seu **API Token**
3. No app: ⚙️ Configurações → campo **Token Brapi** → cole → Salvar
4. Plano gratuito: 1.000 requisições/dia — mais que suficiente

## 🔒 Segurança

- Dados salvos localmente no dispositivo de cada usuário
- Senhas nunca armazenadas em texto puro
- Todas as chamadas externas via HTTPS
- Nenhum dado enviado a servidores de terceiros
- Código 100% aberto e auditável
